UoG Formula SAE Team - Embedded Dash

An onboard dash built for a formula-style EV, designed in collaboration with the embedded subsection of the Gryphon Racing Team. The back-end of the project utilizes C++ to connect various electrical signals from the Energy Meter, BMS, and Motor Controller to a centralized Raspberry Pi. The front-end makes use of QML to display several data configurations, including a Race, Debug, and Tuning mode. The codebase for the project is managed in a professional manner, utilizing Git for organized version control.

Online Multiplayer Game - Teeny Tanks

A browser-based online multiplayer PvP game, created with the free and open-source Godot game engine. The project was coded in GDScript, a language purpose built for the engine with nearly identical syntax to Python. The game establishes a client-server connection using a personal Linux web server. An NGINX server block allows users to connect through the browser using the WebSocket protocol. All project files are properly managed using Git version control.

Skills

  • Python (GDScript)
  • Linux
  • NGINX

3D Printed Kinder Egg Toy - Wall Wrecker

A small toy designed for use in Kinder Surprise eggs, designed in collaboration with a small team of engineers. The toy represents a mobile battering ram, consisting of a wind-up crank powering a torsion spring gearbox. The toy was modelled in SolidWorks, as all members of the group are CSWA certified. The parts were prepared using the Cura slicing software and manufactured through 3D Printing. The finished product, including cost and market research, was presented to a panel of judges for evaluation.

Skills

  • SolidWorks
  • 3D Printing
